Centrage CSS

Voici un petit récapitulatif sur le centrage aussi bien horizontal que vertical en CSS.
Le but de cette page est de lister les différentes techniques existantes pour pouvoir centrer un ou plusieurs block au sein d'un block parent.

Centrage Horizontal

Centrage Horizontal - Simple span

Démonstration

simple span

Description

span simple, le parent a un text-align : center;
Fonctionne en display : inline; et display : inline-block;

Code


			.box__parent {
				text-align : center;
			}

			.box__child {
				width: $size_child_width;
				height: $size_child_height;
				display: inline-block;
				
				padding: $size_child_padding;
			}

Centrage Vertical

Centrage Vertical - Line Height

Démonstration

coucou les amis

Description

"line-height : 300px;" La taille de la ligne est égale à celle du parent.
Cette technique ne marche qu'en : "display: inline;"
Attention : En "display: inline;" il n'est pas possible de définir la taille du fils.

Code


			.box__parent {
				text-align: center;
			}

			.box__child {
				display: inline; 
				line-height : $size_parent_height;
				padding: $size_child_padding; 
			}

Centrage Vertical - Display Table (Vertcal Align)

Démonstration

coucou les amis

Description

On place le fils dans un conteneur
Le parent est en "diplay : table;"
Le conteneur est en "diplay : table-cell;" et "vertical-align : middle"

Code


			.box__parent {
				text-align: center;
				display: table; 
			}

			.box__container {
				display: table-cell;
				vertical-align: middle;
			}

			.box__child {
				width: $size_child_width;
				height: $size_child_width;
				display: inline-block;
				
				padding: $size_child_padding; 
			}

Centrage Vertical - Absolute + bords à 0

Démonstration

coucou les amis

Description

Le fils est en "position: absolute;" et le parent en "position: relative;"
On définit ensuite l'epace des bords (left, right, top, bottom) à 0 ainsi qu'une marge auto.

Code


			.box__parent {
				position: relative; 
			}

			.box__child {
				width: $size_child_width;
				height: $size_child_width;
				display: block;
				position: absolute;
				
				left: 0;
				right: 0;
				top: 0;
				bottom: 0;
				
				padding: $size_child_padding; 
				margin: auto;
			}

Centrage Vertical - Absolute + Margin Négatives

Démonstration

coucou les amis

Description

Le fils est en "position: absolute;" et le parent en "position: relative;"
on position l'élément à 50% de la gauche et du haut
on recentre en mettant une margin négative de la motié de la taille de l'élément (+ éventuel padding)
Attention : Fonctionne que si le fils est en "display: block;"

Code


			.box__parent {
				position: relative; 
			}

			.box__child {
				width: $size_child_width;
				height: $size_child_height;
				display: block;
				position: absolute;
				
				left: 50%;
				top: 50%;

				padding: $size_child_padding; 

				margin-left: -($size_child_width / 2 + $size_child_padding);
				margin-top: -($size_child_height / 2 + $size_child_padding);
			}

Centrage Vertical - Absolute + Translate

Démonstration

coucou les amis

Description

Le fils est en "position: absolute;" et le parent en "position: relative;"
on position l'élément à 50% de la gauche et du haut
on recentre en faisant une translation de -50%

Code


			.box__parent {
				position: relative; 
			}

			.box__child {
				width: $size_child_width;
				height: $size_child_height;
				display: block;
				position: absolute;
				
				left: 50%;
				top: 50%;

				padding: $size_child_padding; 

				transform: translateX(-50%) translateY(-50%);
			}
