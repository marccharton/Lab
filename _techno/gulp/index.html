Tuto Gulp

Dans ce super tuto nous avons utilisé Gulp.js afin d'automatiser quelques tâches ingrates.

Voici une liste de ce qui a été fait :

  • Installation de Gulp
  • Création du fichier de config gulpfile.js
  • Création de la tâche par défaut -> ('default')
  • Création d'une tâche pour la transpilation des fichiers coffeescript -> ('coffee')
  • Création d'une tâche watch pour lancer automatiquement gulp à la savegarde -> ('watch')
  • Utilisation de livereload pour charger automatiquement ce qui a été généré
